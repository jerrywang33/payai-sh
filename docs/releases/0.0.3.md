# PayAI 0.0.3

`v0.0.3` adds runtime validation helpers for PayAI's public data shapes.

## Published Packages

- `@payai-sh/core` v0.0.3
- `@payai-sh/x402` v0.0.3

## Changes

- Added `validateMoney` and `parseMoney`.
- Added `validateSpendingGrant` and `parseSpendingGrant`.
- Added `validatePaymentQuote` and `parsePaymentQuote`.
- Added `validatePaymentReceipt` and `parsePaymentReceipt`.
- Added `PayAIValidationError` with structured `issues`.
- Updated the x402 adapter to validate normalized quotes before policy evaluation.
- Added tests for validation helpers and invalid x402 quote handling.

## Install

```bash
npm install @payai-sh/core @payai-sh/x402
```
