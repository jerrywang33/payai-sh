# PayAI 0.0.2

`v0.0.2` is the first polish release after the initial npm publication.

## Published Packages

- `@payai-sh/core` v0.0.2
- `@payai-sh/x402` v0.0.2

## Changes

- Clarified npm install path in README and docs.
- Added hosted docs pages for architecture, schemas, and agent integration.
- Added `SECURITY.md` and `CONTRIBUTING.md`.
- Fixed the website SDK example to use the actual `createPayAIFetch` API.
- Updated npm package READMEs so npm users see install, docs, and license information.
